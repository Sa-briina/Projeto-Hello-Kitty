# <img src="https://github.com/Sa-briina/Projeto-Hello-Kitty/blob/main/Readme.img/sanrio.webp" width="28" style="vertical-align: middle; margin-bottom: -5px;" /> 🌸 𝒫𝓇𝑜𝒿𝑒𝓉𝑜 𝐻𝑒𝓁𝓁𝑜 𝒦𝒾𝓉𝓉𝓎 — Página Interativa com Tema Sanrio




---

Este projeto é uma página web interativa inspirada no universo encantador da personagem Hello Kitty, desenvolvido como parte do meu aprendizado no curso **Dev em Dobro**. O foco principal foi aplicar e aprofundar conhecimentos em **HTML5, CSS3 e JavaScript**, construindo uma interface responsiva, visualmente atraente e funcional.

Com base em um projeto modelo apresentado pelos meus professores, personalizei todos os aspectos — do design ao comportamento dos elementos — para criar uma experiência única e envolvente. Além disso, utilizei ferramentas modernas como [V0.dev](https://v0.dev/) para explorar efeitos visuais dinâmicos e melhorar a interatividade do site.

---

## <img src="https://github.com/Sa-briina/Projeto-Hello-Kitty/blob/main/Readme.img/seta.webp" width="20" style="vertical-align: middle;" /> Funcionalidades Principais

- Layout responsivo, adaptando-se perfeitamente a diferentes tamanhos de tela  
- Estilo visual inspirado na marca Sanrio, com cores e elementos que remetem à Hello Kitty  
- Animações suaves e interativas que enriquecem a experiência do usuário  
- Cartões detalhados apresentando personagens do universo Hello Kitty  
- Navegação simples e intuitiva, ideal para todos os públicos  

---

## 🛠️ Tecnologias Utilizadas

- <img src="https://github.com/Sa-briina/Projeto-Hello-Kitty/blob/main/Readme.img/html.webp" width="18" /> **HTML5** para a estrutura semântica da página  
- <img src="https://github.com/Sa-briina/Projeto-Hello-Kitty/blob/main/Readme.img/css.webp" width="18" /> **CSS3** com técnicas modernas para layout e efeitos visuais  
- <img src="https://github.com/Sa-briina/Projeto-Hello-Kitty/blob/main/Readme.img/js.webp" width="18" /> **JavaScript** para interatividade e manipulação dinâmica dos elementos  
- <img src="https://github.com/Sa-briina/Projeto-Hello-Kitty/blob/main/Readme.img/git.webp" width="18" /> **Git e GitHub** para controle de versão e hospedagem  
- **V0.dev** como ferramenta de pré-visualização e prototipação rápida
